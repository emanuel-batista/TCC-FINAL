1) CRIAR NAVBAR LATERAL DO DASHBOARD PARA MELHORAR NAVEGAÇÃO
2) ARRUMAR BANCO DE DADOS COM FOREIGN KEY EM TODAS AS TABELA(COM EXCESSÃO A TABELA USUÁRIO), COLOCAR OS CAMPOS DO NOME DE USUARIO E FOTO DE PERFIL DO USUÁRIO SÃO FOREIGN KEYS(talvez usar inner join)
3) ARRUMAR CÓDIGOS POSTERIORMENTE DO ADM COM SESSÃO PARA PUXAR O ID DO USUARIO, NOME E FOTO DE PERFIL
4) FAZER OS BAGULHOS DO FRONT DO JEITO MAIS FÁCIL E PRÁTICO POSSÍVEL, PEGAR ELEMENTOS JÁ PRONTO DO BOOTSTRAP, SÓ SEGUIR OS SEGUINTES REQUERIMENTOS:
    - Página para as notícias, subdivididas em categorias(ex: public/noticias/volei);
    - Página para as entrevistas, seguindo os mesmos exemplos acima.
